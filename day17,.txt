Q:what is data, field, record, database?
 
What is data?
Raw facts or figures without context.
ex:101,ravi,50000,hyb
what is field?
smallest unit of data in a database(column/attribute)
ex: Empname ,EmpID, EmSal , EmpAddress
what is fields?
collection of related fields(row/tuple)
e.g.:(111,ravi,60000,hyd)

what is database?
organized collection of related records stored together.
eg: an employee database containing all employee records.

Data models in DBMS:
=======================
Deff: a data model defines how data is stored organized and manipulated in database.


TYPES OD DATA MODELS:
=======================

1.Pysical data model:
------------------------
def: Describe how data is physically stored in storage devices.
Audience: database administrators

2.Logical Data Model
------------------------
def: defines the structure of data(entities ,attributes, relationships) independent of physical storage areas.
Audience: Developers and data architects.

3.Hierarical Data Model:
------------------------
Def: data is represented as tree structure(parent child relationship)
eg: company-->department--> employees

4.Network Data Model:
------------------------
Data is represented as graph with records(nodes) and relationships(edges)
ex: student enrolled in multiple courses, courses taught by multiple professors.

5.Relation Data Model:
-------------------------
data is stored in tables(relations) with rows(tuples) and columns(attribute)

ex: student(id,name,marks,age,dept_id)
department(dept_id, dep_name)
6.Entity Relationship (E-R model):
----------------------------------
a conceptual model that shows entities(things),attributes(properties) and relationships
ex: student and course
entites of student-->(student_name, student_id,age)
entites of courses-->(course_name, course_id)
